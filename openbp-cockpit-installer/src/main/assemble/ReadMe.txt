------------------------------------------------------------
Welcome to OpenBP @AppVersion@!
------------------------------------------------------------

This installation program will install OpenBP's the graphical
modeling environment (the OpenBP Cockpit) on your computer.


------------------------------------------------------------
Status
------------------------------------------------------------

Incompatible changes in the API can be expected until OpenBP version 1.0.
The ChangeHistory.txt file lists the deltas to older versions.


------------------------------------------------------------
Documentation
------------------------------------------------------------

Currently, there is no up-to-date documentation for the OpenBP Cockpit yet.

The API documentation can be found in the apidoc sub directory of the OpenBP source distribution.
For sample source code, see the JUnit Tests at
	openbp-server/src/test/java/org/openbp/server/test
e. g. the WorkflowTaskTest class.
