------------------------------------------------------------
Thank You for Downloading the OpenBP Source Distribution!
------------------------------------------------------------

The OpenBP server jar files and OpenBP Cockpit installer
is part of the OpenBP binary distribution.


------------------------------------------------------------
Documentation
------------------------------------------------------------

Currently, there is no up-to-date documentation for the OpenBP Cockpit yet.

The API documentation can be found in the apidoc sub directory of the binary distribution.


------------------------------------------------------------
Status
------------------------------------------------------------

Incompatible changes in the API can be expected until OpenBP version 1.0.
The ChangeHistory.txt file lists the deltas to older versions.


------------------------------------------------------------
Support & Contact
------------------------------------------------------------

The project web site is currently being extended by a TRAC instance for OpenBP (see org.edgewall.trac).

For help or suggestions, please contact info@openbp.org.
